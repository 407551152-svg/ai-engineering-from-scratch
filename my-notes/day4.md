## 神经网络是什么

神经网络可以理解为一组可以被训练的参数。

它接收输入数据，然后经过一层或多层计算，最后输出预测结果。

简单结构：

```text
输入层 -> 隐藏层 -> 输出层
```

每一层里都有：

- 权重 `weights`
- 偏置 `bias`
- 激活函数 `activation function`

神经网络训练的目的，就是不断调整权重和偏置，让预测结果越来越接近真实答案。

## 前向传播是什么

前向传播就是从输入计算输出的过程。

在今天的项目里，流程是：

```text
输入 X
-> 乘以第一层权重 W1，加上偏置 b1
-> 经过 sigmoid 激活函数
-> 乘以第二层权重 W2，加上偏置 b2
-> 再经过 sigmoid
-> 得到最终预测 y_pred
```

对应代码：

```python
z1 = np.dot(X, W1) + b1
a1 = sigmoid(z1)

z2 = np.dot(a1, W2) + b2
y_pred = sigmoid(z2)
```

## 损失函数是什么

损失函数用来衡量模型预测错了多少。

如果预测结果和真实答案差距很大，loss 就大。

如果预测结果接近真实答案，loss 就小。

今天项目使用的是 Binary Cross Entropy，因为 AND 逻辑是二分类问题。

```python
loss = binary_cross_entropy(y, y_pred)
```

我对损失函数的理解：

```text
损失函数就是模型的错误分数。
训练的目标就是让这个错误分数越来越低。
```

## 反向传播是什么

反向传播就是从输出结果倒着计算每个参数应该怎么改。

前向传播回答：

```text
模型现在预测成什么？
```

反向传播回答：

```text
模型错在哪里？每个权重应该往哪个方向调整？
```

简单理解：

```text
预测错了 -> 计算错误 -> 倒着分配责任 -> 修改参数
```

## 梯度下降是什么

梯度下降是更新参数的方法。

它会根据反向传播算出来的梯度，调整权重和偏置。

代码形式：

```python
W -= learning_rate * gradient
```

其中：

- `W` 是权重
- `learning_rate` 是学习率
- `gradient` 是梯度

学习率太大，训练可能不稳定。

学习率太小，训练会很慢。

## 激活函数是什么

激活函数让神经网络具备非线性表达能力。

如果没有激活函数，多层神经网络本质上还是线性计算，表达能力会很弱。

今天项目使用的是 sigmoid：

```python
def sigmoid(x):
    return 1 / (1 + np.exp(-x))
```

sigmoid 的输出范围是：

```text
0 到 1
```

所以它很适合二分类任务。
