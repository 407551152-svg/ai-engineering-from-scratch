# Phase 04 Computer Vision 学习笔记

## 图像在计算机中是什么

图片在计算机中本质上是数字数组。

RGB 图片通常是：

height x width x channels

## 像素、通道、尺寸是什么

像素是图片中最小的显示单位。
通道表示颜色维度。
RGB 图片有 3 个通道：
Red
Green
Blue
灰度图只有 1 个通道，表示亮度。

## 卷积是什么

卷积是用一个小矩阵在图片上滑动，对局部区域进行计算。
这个小矩阵叫卷积核。
不同卷积核可以得到不同效果：
边缘检测
模糊
锐化
特征提取

## CNN 为什么适合图像

CNN 适合图像，因为图像具有局部结构。
比如一个边缘、角点、纹理，通常都只和周围一小块像素有关。
CNN 的卷积核可以学习这些局部特征。
图像分类是什么
图像分类就是输入一张图片，模型输出它属于哪个类别。
例如：
输入：一张猫的图片
输出：cat

## 迁移学习是什么

迁移学习是使用已经训练好的模型，在自己的任务上继续使用或微调。
它的好处是：
不需要从零训练大模型
需要的数据更少
训练速度更快
效果通常更好
