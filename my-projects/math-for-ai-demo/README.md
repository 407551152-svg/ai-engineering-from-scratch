# Math for AI Demo

这是一个用于理解 AI 数学基础的小项目。

项目通过 Python 实现向量点积、向量长度和余弦相似度，用来演示 AI 系统中常见的“相似度计算”思想。

## 项目目标

理解下面几个概念：

- 向量 Vector
- 点积 Dot Product
- 向量长度 Magnitude
- 余弦相似度 Cosine Similarity
- 为什么相似度计算可以用于搜索、推荐和 RAG

## 项目功能

程序会：

1. 定义一个查询向量
2. 定义几篇文档对应的向量
3. 计算查询向量和每篇文档的相似度
4. 找出最相关的文档

## 使用技术

- Python
- math 标准库
- 基础线性代数
